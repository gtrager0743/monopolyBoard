# Monopoly Board Simulator (Spring 2026)

## Overview
A C++ simulation of a Monopoly board using a **Circular Linked List**. This project demonstrates manual pointer management, memory safety, and traversal logic without using arrays or indexing shortcuts.

## Constraints
* **Max Capacity:** 40 Spaces (Hard limit).
* **Structure:** Circular Linked List (Tail points to Head).

## Build Instructions
* **Compile:** `g++ -std=c++17 main.cpp -o monopoly`
* **Run:** `./monopoly`

## Features
* [ ] **Core A:** Add Space (Capacity Enforcement)
* [ ] **Core B:** Bulk Add
* [ ] **Core C:** Player Movement (Traversal)
